# Post de Ejemplo para V1tr0

Este es un post de ejemplo para mostrar cómo funciona MDX en tu blog. MDX te permite utilizar componentes React dentro de tu contenido Markdown.

## ¿Qué es MDX?

MDX es una extensión de Markdown que te permite usar componentes JSX en tu contenido. Esto te permite crear contenido rico e interactivo para tu blog.

## Ejemplos de formato en Markdown

### Listas

Aquí hay una lista ordenada:

1. Primer elemento
2. Segundo elemento
3. Tercer elemento

Y una lista no ordenada:

- Un elemento
- Otro elemento
- Un elemento más

### Código

Puedes incluir bloques de código:

```javascript
function saludar(nombre) {
  return `¡Hola, ${nombre}!`;
}

console.log(saludar("mundo"));
```
