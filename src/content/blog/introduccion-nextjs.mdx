# Introducción a Next.js

Next.js es un framework de React que permite funcionalidades como renderizado del lado del servidor y generación de sitios estáticos para aplicaciones web basadas en React.

## ¿Por qué usar Next.js?

Next.js proporciona una experiencia de desarrollo excepcional con todas las funciones que necesitas para la producción:

- **Renderizado híbrido**: Combina generación estática (SSG) y renderizado del servidor (SSR)
- **Soporte TypeScript**: Integración completa con TypeScript
- **Optimización inteligente**: Optimización automática de imágenes, fuentes y más
- **Recarga rápida**: Hot reload para una experiencia de desarrollo fluida
- **Enrutamiento basado en archivos**: Sistema de enrutamiento intuitivo basado en la estructura de archivos
