# Fundamentos del Deep Learning y las Redes Neuronales

## ВҝQuГ© es Deep Learning?

El **Aprendizaje Profundo** o **Deep Learning** es una subdisciplina del aprendizaje automГЎtico que se distingue por el uso de algoritmos basados en **redes neuronales profundas**. Estas redes replican, de manera conceptual, la interconexiГіn de las neuronas biolГіgicas en el cerebro humano.

Como seГұala VerГіnica HernГЎndez (BBVA AI Factory):

> "El 'deep learning' se centra en crear algoritmos que pueden aprender a travГ©s de redes neuronales profundas"

Las redes neuronales profundas crean una jerarquГӯa de conceptos donde las representaciones mГЎs abstractas se construyen a partir de otras mГЎs simples.

## Estructura de una Red Neuronal Artificial

Una **red neuronal artificial** es un modelo computacional compuesto por nodos interconectados, denominados "neuronas", que trabajan en conjunto para procesar y analizar datos.

**MatemГЎticamente:** Una red neuronal transforma un vector de entradas en un vector de salidas mediante operaciones matemГЎticas organizadas en capas.

## Componentes MatemГЎticos de una Neurona

### 1. CaracterГӯsticas de entrada (X)

Vector de entrada: **X вҲҲ в„қвҒҝ**


### 2. Pesos (W)

Matriz que pondera la importancia de cada conexiГіn.

### 3. Sesgos (b)


## La FunciГіn de ActivaciГіn Sigmoide

La operaciГіn central de una neurona involucra una **funciГіn de activaciГіn no lineal**.

### FunciГіn Sigmoide Пғ(z)

**FГіrmula:**
```
Пғ(z) = 1 / (1 + eвҒ»б¶»)
```

**Comportamiento:**
- Entrada grande y positiva вҶ’ salida вүҲ 1 (neurona activa)
- Entrada grande y negativa вҶ’ salida вүҲ 0 (neurona inactiva)
- Rango de salida: [0, 1]

```python
import numpy as np

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

# Ejemplo
z = np.array([-2, 0, 2])
print(sigmoid(z))  # [0.119, 0.5, 0.881]
```

### Ajuste con ParГЎmetros

En la expresiГіn **Пғ(3(zвҲ’5))**:
- Factor **3**: Intensifica la pendiente (transiciГіn mГЎs abrupta)
- Desplazamiento **-5**: Mueve el punto de transiciГіn hacia la derecha

## Redes Multicapa

En una red multicapa, las neuronas se organizan en capas sucesivas:

```
Capa Entrada вҶ’ Capa Oculta 1 вҶ’ Capa Oculta 2 вҶ’ Capa Salida
```

### FГіrmula de ActivaciГіn

```
aбөў[ЛЎ] = Пғ(ОЈ(wбөўвұј[ЛЎ] В· aвұј[ЛЎвҒ»В№]) + bбөў[ЛЎ])
```

**Donde:**
- **aвұј[ЛЎвҒ»В№]**: ActivaciГіn de la neurona j de la capa anterior
- **wбөўвұј[ЛЎ]**: Peso de conexiГіn
- **bбөў[ЛЎ]**: Sesgo de la neurona i

```python
# Ejemplo conceptual
def calcular_activacion(a_anterior, W, b):
    z = np.dot(W, a_anterior) + b
    return sigmoid(z)
```
